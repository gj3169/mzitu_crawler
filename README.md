### 依赖

- **python3.5**
- requests
- BeautifulSoup
### 使用方法

安装好上面提到的依赖之后，命令行执行
```
python spidermzitu.py
```
### 功能

抓取妹子图-性感妹子目录下的所有图片，并存储在`E:\mzitu`文件夹下，如需修改此文件夹，可以修改代码中的目录。
